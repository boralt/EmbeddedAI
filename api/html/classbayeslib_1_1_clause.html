Detailed Description

Clause is one instance/occurance for VarSet (subset of domain variables) it assign true/false value for each variable in VarSet Internally, the Clause is defined by integral InstanceId value in which the states of all Variables in the varset are binary 0 or 1 placed according to offset inside the VarSet. Therefore numeric value of InstanceId is only significant for Clauses constructed from identical VarSet
